1.  Criar usuário no Git Hub
2.  Criar repositório
3.  Criar diretório no computador
4.  Download e instalação do Git
5.  Copiar o caminho do repositório no Git Hub
6.  Executar o comando dentro do diretório criado no PC -> git clone + colar caminho copiado
7.  No VSCode, abrir um novo terminal
8.  Verificar os arquivos alterados -> git status
9.  Gerar o pacote de atualização -> git add .
10. Verificar o conteúdo do pacote -> git status
11. Configurar o seu email -> git config --global user.email "cesar.augusto.moraes@hotmail.com"
12. Configurar o seu nome -> git config --global user.name "Cesar Augusto Moraes"
13. Enviar o conteúdo do pacote para o Git -> git commit -m "alterando e incluindo arquivos"
14. Efetivar as atualizações no Git -> git push
